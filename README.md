# react-frontend-map

React Frontend - City Map Population.

This project is using:
- Mapbox
- React Router
- React Hook Form
- Axios
- Styled Components

## Building and running on localhost

First install dependencies:

```sh
npm install
```

## Running

```sh
npm run start
```

## Production
To create a production build:

```sh
npm run build
```
